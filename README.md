## Project
  **NETFLIX USER ANALYSIS - Exploratory Data Analysis (EDA)**
  
OBJECTIVE:

  To explore and analyze the Netflix datset to uncover key insights about content types , trends over time ,countries, genres and user ratings. 

## Tools Used :
  1. python
  2. pandas (for data cleaning & manipulation)
  3. matplotlib ( for visualization)
  4. seaborn (for visualization)

## Data Exploration 
  Performed various analyses to answer interesting questions like :
- What type of content (Movie or TV show ) dominate the platform ?
- Which country produces the most content ?
- What are the most popular genres ?
- How has NETFLIX content evolved pver the years?

## Data Visulization
  Used Seaborn & Matplotlib to plot :
  - Top countrise producing Netflix content
  - content distribution over years
  - most common rating
  - content type( Movie vs. Tv show)

## Key Insights
  - **MOVIES** dominate the platform.
  - The **UNITED STATES** is the largest producer of Netflix content.
  - Netflix added the most content in **2019**.
  - Common rating include **TV_MA** ,**TV_14** & **R**.
  - Genres like **Dramas**,**Comedies** & **Documentaries** are very popular.
